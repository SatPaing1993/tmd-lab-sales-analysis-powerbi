# TMD Lab Sales Analysis Dashboard

## 📊 Project Overview
This project presents an interactive Power BI dashboard designed to analyze laboratory performance, including test demand, revenue trends, and referral sources.

## 🎯 Objectives
- Analyze most requested lab tests
- Identify top revenue-generating tests
- Track monthly sales trends
- Understand referral distribution

## 📌 Key Features
- KPI metrics: Total Sales, Refer Fees, Test Count, Patient Count
- Visualization of most requested tests
- Revenue analysis by test type
- Monthly sales trend analysis
- Referral source breakdown
- Interactive filters (Date, Doctor, Test Name)

## 🔍 Key Insights
- CPA is the top revenue-generating test
- Sales dropped significantly in April (~36%)

## 🛠 Tools Used
- Power BI
- Microsoft Excel
